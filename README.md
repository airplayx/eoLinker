## 简介

**eoLinker是国内最大的在线API接口管理平台，提供自动生成API文档、API自动化测试、Mock测试、团队协作等功能，旨在解决由于前后端分离导致的开发效率低下问题。**

目前eoLinker为来自全球的超过2万家企业提供快速、专业、稳定的API管理服务。eoLinker是Google谷歌开发者联盟的合作产品与企业，不定期举办线下交流分享活动促进国内API管理领域的发展。

# eoLinker API 测试增强插件（自动化测试插件）
eoLinker提供了非常强大的在线API接口测试功能，在使用之前，请先安装 **eoLinker自动化测试增强插件** ，该插件为您提供了以下功能支持：
1. 线上网站、跨域、本地测试，支持通过远程网站访问本地localhost进行测试
2. 支持文件类型的请求参数
3. 支持更多请求协议
4. 支持简易、高级自动化测试
5. 支持全局变量等需要插件支持的测试

插件获取地址：本项目内 eo-chrome-extension 文件夹即为扩展程序文件夹

插件仅支持Chrome。

---

#### 安装

**Chrome：**

1.打开扩展程序或地址栏输入回车 chrome://extensions

2.开启开发者模式 -> 加载已解压的扩展程序 -> 选中打开的 extension 文件夹
